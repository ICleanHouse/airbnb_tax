# assets/

Brand marks for Host Cleaners. All SVG — crisp at any size.

| File | Use |
|---|---|
| `mark.svg` | **Primary mark & favicon** — house-with-sparkle in the coral app tile. |
| `favicon.svg` | Same as `mark.svg`, named for `<link rel="icon">`. |
| `glyph.svg` | Coral house, transparent background, sparkle knocked out — inline use on light surfaces. |
| `glyph-mono.svg` | Single-color (ink) glyph — stamps, watermarks, one-color print. Set `fill` to `currentColor` to inherit text color. |
| `logo.svg` | Horizontal lockup: mark + “Host Cleaners” wordmark (Geist 650). |

Wire the favicon:
```html
<link rel="icon" type="image/svg+xml" href="assets/favicon.svg">
```

The mark is **original to Host Cleaners** — a house containing a sparkle (“a
sparkling-clean home”), intentionally distinct from Airbnb's looping “Bélo.”
The coral is the brand color (`#ff385c`); see the root `README.md` → Iconography.
No raster/photographic brand assets exist yet — add them here if produced.
